# 1-divide

This project implements a method that divides all elements of a matrix.

## MyMath.Matrix.Divide

Divides all elements of a matrix by a given number.

### Usage
```csharp
int[,] matrix = { { 4, 8 }, { 12, 16 } };
int[,] result = MyMath.Matrix.Divide(matrix, 2); // Returns { { 2, 4 }, { 6, 8 } }
```
